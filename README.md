# referent
референт текстов с ИИ

## Установка и запуск

### Установка зависимостей

```powershell
pnpm install
```

### Запуск в режиме разработки

```powershell
pnpm dev
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

### Сборка для production

```powershell
pnpm build
```

### Запуск production версии

```powershell
pnpm start
```

### Линтинг

```powershell
pnpm lint
```
